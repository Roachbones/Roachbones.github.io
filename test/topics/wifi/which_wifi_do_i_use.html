FAQ - Wi-Fi Topics - Which Wi-Fi do I use?

Which Wi-Fi Network to Use

  • Most devices (laptops, desktops, and phones) should use Marietta College. When prompted to enter credentials, use your MyMarietta username and password.
  • If you're in a residence hall and setting up a TV, Roku, Fire Stick, or similar device that doesn't move, connect it to MC Streaming.
  • MC Guest is for visitors without Marietta College credentials.

If you have difficulty joining a network, text a ticket to 740-530-4376 and we'll assist you.
